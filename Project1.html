项目1-智能型打气泵
       课题设计的基本内容与拟解决的主要问题有：挑选单片机型号，合理分配IO口功能，特别是大电流负载的驱动画出原理图并设计电路板。编写程序仿真测试、编写上位机软件。实物测试，实现电路板和上位机交互数据显示和界面操作控制系统。单片机型号和IO口分配同时实现的功能设置不同IO口的驱动能力并严格控制好时序，电路板LCD气压显示设计、编写上位机界面的布局合理，最后实现计算机与多个单片机的多机通信。设计图纸规范、代码设计和注释的规范。
       智能打气泵主要是面向国外用户的产品。目前国内智能打气泵市场比较小，很少有人会选择购买，主要原因是国内人口密度大，道路好，很少遇到车胎爆了没法去修理厂的问题。而国外地广人稀，车胎在路上爆了就只能等待援救人们更需要打气泵来解决打气问题，受自由主义影响国外的人更喜欢自己解决问题。所以会选择购买智能打气泵，从而智能打气泵在国外的市场比国内大，南方泰科的产品也大多最终是面向国外用户的。国内产品有竞争压力但并不是特别大，属于高速发展中的产业。综上所述智能打气泵在全世界范围内有着巨大的商业需求，并且行业竞争环境比较良性。
上位机操作界面
       软件设计主要运用 Keil 进行编程， C51 是美国 Keil Software 公司出品的 51 系列兼容单片机 C 语言软件开发系统，与汇编相比， C 语言在功能上、结构性、可读性、可维护性上有明显的优势，因而易学易用。 Keil 提供了包括 C 编译器、宏汇编、链接器、库管理和一个功能强大的仿真调试器等在内的完整开发方案，通过一个集成开发环境（μ Vision ）将这些部分组合在一起。运行 Keil 软件需要 WIN98 、 NT 、 WIN2000 、 WINXP 等操作系统。如果你使用 C 语言编程，那么 Keil 几乎就是你的不二之选，即使不使用 C 语言而仅用汇编语言编程，其方便易用的集成环境、强大的软件仿真调试工具也会令你事半功倍。
